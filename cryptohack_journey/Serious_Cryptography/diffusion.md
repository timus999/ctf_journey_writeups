- **Purpose**: Spread the influence of each **plaintext** bit over many ciphertext bits, hiding statistical patterns. 
    
- **Mechanism**: Achieved through **permutation** or **transposition** (e.g., permutation layers, MixColumns in AES). 
    
- **Effect**: Changing one bit of plaintext should change approximately **half** of the ciphertext bits (avalanche effect). 
    
- **Used in**: Primarily **block ciphers**.
    

> Example: In a block cipher, flipping one plaintext bit results in widespread changes across the ciphertext block
